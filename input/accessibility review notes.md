## A. Mandatory Corrections

### A-1 🔴 Contrast ratio, Gallery 3 section panel
The Crossroads Gallery introduction panel design (v3.2 layout) uses dark bronze text (#5C4326) on the terracotta background (#C97B4F) approved in the design palette. Measured contrast ratio is 3.1:1, below the 4.5:1 minimum required for body copy under Style Guide §6. **Action:** Design to darken background or lighten text by next proof round. This panel's word-count exception (Style Guide §12, item 1) does not extend to a contrast exception — these are separate provisions and should not be conflated in vendor conversations.

### A-2 🔴 Label mounting height, freestanding pedestal captions
Pedestal captions for *Echo Chamber* and *Reliquary (after Benin)* (see `object_captions.md`) are shown in the current mockup mounted at 52 inches centerline. Style Guide §6 requires 46–48 inches centerline. **Action:** Design to lower both pedestal mounts by 4–6 inches. Confirm sightline is not obstructed by the sculptures' own bases at the corrected height.

### A-3 🔴 Missing large-print entries
The large-print binder draft is missing entries for object labels 2.3 (Benin plaque, 1975.3.9) and 3.5 (Kōga scroll, 2004.22.6). Both objects carry longer-than-standard interpretive text (per Style Guide §12, items 2–3), and the binder must include the full approved text, not a truncated version. **Action:** Interpretation to supply final large-print-formatted text to the bindery by August 1.

### A-4 🔴 Educational panel mounting height
`educational_panels.md` notes a "lowered rail height (38–42 in.)" as a production note but the current floor plan shows only one mounting height for all five family panels, matching the standard 46–48 inch adult centerline. As written, the family panels do not meet the Museum's stated commitment to child-accessible mounting. **Action:** Design to confirm whether dual-height mounting is in the current budget; if not feasible for all five, prioritize Panels 1, 2, and 5, which are in higher-traffic, wider-aisle locations, for dual-height installation.

### A-5 🔴 Alt text/audio parity gap
No audio guide script has been drafted for Panel 2 ("Try This — Spot the Echo"), which relies on a visitor physically locating two objects in two different galleries. This is a visual/spatial task not currently addressed by an equivalent audio-guide accommodation. **Action:** Interpretation and Audio Guide vendor to develop a verbal wayfinding equivalent (e.g., "the small brass object in the third case to your right") before the audio guide script is finalized.

## B. Recommended (Non-Blocking)

### B-1 🟡 Reading level, object label 3.4 (Vanitas still life)
This label scores at grade 9.1 (Flesch-Kincaid), above the grade 7–8 target in Style Guide §6 for standard object labels. The elevated score is driven largely by the long final sentence beginning "Vanitas paintings flourished in the Dutch Republic..." **Recommendation:** Consider splitting into two sentences. Not blocking, as Style Guide §6 permits some panels to run to grade 9 with justification; however, this is an object label rather than a section panel, so the higher-grade allowance technically applies to a different category of text. Flagging for Curatorial's awareness rather than requiring a rewrite.

### B-2 🟡 Tactile object candidate not yet identified for Gallery 2
Style Guide §6 calls for at least one tactile-reproduction or touchable-material candidate per gallery. Galleries 1, 3, and 4 each have a designated tactile element in the current plan (shabti replica, goldweight replica, and a bronze swatch sample from Whitfield-Marsh's studio, respectively). Gallery 2 does not yet have one identified. **Recommendation:** Consider a champlevé enamel swatch sample paired with the reliquary casket label, subject to Conservation approval of a suitable reproduction material.

### B-3 🟡 Wayfinding "ECHO" floor markers — color-only signal risk
The current floor marker design uses a color shift (terracotta to gold) as the primary way of indicating an "echo" pairing. Style Guide does not currently require markers to avoid color-only signaling, but WCAG general guidance recommends against relying on color alone to convey meaning. **Recommendation:** Add the word "ECHO" in text on the marker itself (already planned per `object_captions.md`) so the pairing is legible without reliance on color perception — this appears to already be the intended design; confirming it is reflected in the current floor-graphic proof, which as of July 10 shows the color treatment only, without text overlay yet.

### B-4 🟡 Extended label for object 2.3 — plain-language pass
The provenance note approved under Style Guide §12, item 2, for the Benin plaque is factually precise but written at a higher reading level than the rest of the label (grade 10.8 for that added paragraph specifically). Because this content addresses a sensitive and important topic, **recommend** a plain-language pass that preserves all factual content and does not soften or omit the substance, rather than a word-count reduction. This is a recommendation for clarity, not a request to shorten or alter the substance of the approved note.
